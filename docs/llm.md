# Large Language Models (LLMs)

A Large Language Model (LLM) is a machine learning model trained on massive
amounts of text to understand and generate human-like language. Modern LLMs
are based on the Transformer architecture, which uses self-attention to model
relationships between words.

Common capabilities: text generation, summarization, translation, question
answering, and code generation.

LLMs are accessed through APIs (OpenAI, Anthropic, Google) or run locally
with tools like Ollama. They have a fixed "context window" - the amount of
text they can consider at once - which matters when designing RAG systems.
